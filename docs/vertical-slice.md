# Vertical Slice Definition

## 目标

最小垂直切片需要覆盖你们需求文档里最核心的玩家体验路径，而不是只生成一堆空文件。

## 流程

1. 进入主菜单，看到 `Start / Load / Settings / Exit`
2. 点击 `Start` 进入一段示例剧情
3. 出现背景、人物占位图、对话框与 HUD
4. 对话按逐字显示推进
5. 左键在逐字显示时补完当前句子，再次左键推进下一句
6. 右键隐藏 HUD / 对话 UI
7. 右上角能打开历史记录、切换自动播放
8. 底部能打开 SAVE / LOAD / CONFIG / EXIT
9. SAVE / LOAD 至少支持 50 个槽位（5 页 * 10 槽位）
10. 覆盖已有存档前弹出确认框
11. 配置界面可调节 BGM / Voice / UI 音量，以及自动播放速度和文本速度
12. 示例剧情包含背景转场、角色移动、BGM 与配音调用钩子

## 占位资源策略

在没有正式素材的前提下，phase 1 使用：

- `Solid()` 作为背景占位
- `Text()` 或纯色块作为人物占位
- `renpy.loadable()` 包裹音频调用，保证缺少资源时项目仍可生成和静态测试

## 验收重点

- screen 名称与关键交互钩子存在
- 文本速度 / 自动播放速度配置值能映射到运行时变量
- 右键隐藏 UI 的行为被显式定义
- 生成结果稳定、可重复
